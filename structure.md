## Структура страницы города

### Правила

1. **Обязательные разделы** — присутствуют всегда, даже если информации мало (отметить как «нет данных»).
2. **Опциональные разделы** — включаются только при наличии содержательной информации.
3. **Раздел «Однодневная поездка»** — добавляется только если страница создаётся для конкретной поездки; для общего справочника — опускается.
4. **Порядок разделов** — фиксированный (см. шаблон ниже); не переставлять без обновления этого файла.
5. **Таблицы** — предпочтительный формат для списков мест с атрибутами (часы, цена, рейтинг, время посещения).
6. **Язык** — всё на русском; исключение: устоявшиеся английские названия заведений, улиц, мероприятий.
